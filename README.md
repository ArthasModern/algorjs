# algorjs
Algorithm library implemented with JavaScript

## Sort
    实现列举了三种最为通用的排序算法：
    mergeSort ---- 归并排序
    heapSort  ---- 堆排序
    quickSort ---- 快速排序

    并扩展为可以自定义比较方法`compFunc`
    通过自定义比较方法，可以自由实现对任意数据结构的比较排序；
    比如，按年岁`man.age`排序；
    比如，正序、倒序、自定义优先级排序等等；
